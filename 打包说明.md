# 课表小工具 - 打包说明

## 方法一：使用 build_exe_simple.py（推荐）

1. **安装依赖**
```bash
pip install pyinstaller pillow
```

2. **运行打包脚本**
```bash
python build_exe_simple.py
```

3. **等待打包完成**
- 脚本会自动创建图标
- 打包完成后exe位于 `dist/课表小工具.exe`

4. **创建桌面快捷方式**（可选）
- 打包完成后按提示输入 `y` 即可创建桌面快捷方式

## 方法二：手动打包

```bash
# 安装pyinstaller
pip install pyinstaller

# 打包命令
pyinstaller --name=课表小工具 --windowed --onefile --icon=icon.ico --add-data="icon.ico;." schedule_widget.py
```

## 打包后的文件

打包完成后会生成：
- `dist/课表小工具.exe` - 主程序
- `build/` - 临时文件（可删除）
- `课表小工具.spec` - 打包配置

## 运行方式

### 方式1：直接运行exe
双击 `dist/课表小工具.exe` 即可启动

### 方式2：创建快捷方式
1. 右键点击exe文件
2. 选择"发送到" -> "桌面快捷方式"

### 方式3：使用 start.vbs（无窗口启动）
双击 `start.vbs` 文件启动

## 任务栏图标

打包后的exe会在任务栏显示图标：
- 运行时图标会出现在任务栏
- 可以点击任务栏图标选中程序
- 可以固定到任务栏方便下次启动

## 注意事项

1. **图标不显示？**
   - 确保 `icon.ico` 文件存在
   - 尝试将exe复制到其他目录再运行
   - Windows可能需要刷新图标缓存

2. **杀毒软件误报？**
   - PyInstaller打包的程序可能被误报
   - 将exe添加到杀毒软件白名单
   - 或使用 `--noupx` 参数打包

3. **缺少依赖？**
   - 确保所有import的库都已安装
   - 使用 `pip install -r requirements.txt` 安装依赖

## 自定义图标

将你自己的 `icon.ico` 文件放在目录中再打包，即可使用自定义图标。

图标要求：
- 格式：ICO
- 建议尺寸：256x256 或包含多尺寸（16,32,48,256）
