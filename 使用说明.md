# 桌面课表小工具 v1.2

## 功能特点

1. **显示当天课程** - 自动根据系统日期显示当天的课表
2. **倒计时功能** - 显示距离下一节课还有多长时间
3. **当前课程高亮** - 正在上课的课程用绿色高亮显示
4. **课程状态标识** - 已结束(✓)、进行中(▶)、即将到来(⏳)
5. **透明悬浮窗** - 可以置顶显示在桌面，支持拖动位置
6. **完整课表** - 查看和编辑整周课程
7. **自定义外观** - 支持背景图片、字体颜色、描边等
8. **自由调整大小** - 支持8个方向拉伸窗口

## 快速启动

### 方式1：EXE程序（推荐）

```bash
# 1. 打包生成exe
python build_exe_simple.py

# 2. 运行 dist/课表小工具.exe
# 可将exe固定到任务栏
```

### 方式2：Python脚本

```bash
# 安装依赖
pip install -r requirements.txt

# 运行程序
python schedule_widget.py
```

### 方式3：无命令框启动

双击运行 `start.vbs` 或 `start_hidden.bat`

## 使用方法

### 窗口操作

| 操作 | 说明 |
|------|------|
| 拖动标题栏 | 移动窗口位置 |
| 边缘拉伸 | 将鼠标移到窗口边缘，拖动调整大小 |
| 右键 | 显示菜单（完整课表、设置、置顶、退出） |
| 滚轮 | 滚动查看课程列表 |
| — 按钮 | 最小化窗口 |
| ✕ 按钮 | 关闭程序 |

### 课程编辑

在"完整课表"窗口中：
- **添加**：点击每天的"➕ 添加"按钮
- **编辑**：点击课程旁的"编辑"按钮
- **删除**：点击课程旁的"删除"按钮

### 自定义设置

在"设置"窗口中可配置：
- **窗口永远置顶** - 窗口始终显示在最前面
- **课表颜色** - 自定义课程列表字体颜色
- **倒计时颜色** - 自定义倒计时字体颜色
- **字体描边** - 添加描边效果，可自定义描边颜色
- **背景图片** - 设置自定义背景图片

## 课表文件格式

课表使用 JSON 格式存储，示例结构：

```json
{
  "1": [
    {"name": "高等数学", "room": "A101", "start": "08:00", "end": "09:35"},
    {"name": "大学英语", "room": "B205", "start": "10:00", "end": "11:35"}
  ],
  "2": [],
  "3": [],
  "4": [],
  "5": [],
  "6": [],
  "7": []
}
```

- `1` = 周一，`7` = 周日
- `name` = 课程名称
- `room` = 教室位置
- `start`/`end` = 开始/结束时间（HH:MM格式）

## 常见问题

**Q: 任务栏没有显示图标？**
A: 使用exe版本，或确保已安装pywin32库：`pip install pywin32`

**Q: 字体描边不显示？**
A: 需要安装Pillow库：`pip install Pillow`

**Q: 背景图片不显示？**
A: 确保图片格式支持（PNG、JPG等），且路径正确

**Q: 如何固定到任务栏？**
A: 1) 运行程序 2) 右键点击任务栏图标 3) 选择"固定到任务栏"

## 依赖安装

```bash
pip install -r requirements.txt
```

依赖列表：
- Pillow >= 9.0.0（图片处理、字体描边）
- pyinstaller >= 5.0.0（打包exe）
- pywin32 >= 227（Windows任务栏支持）

## 注意事项

- 第一次运行会自动生成示例课表 `schedule.json`
- 时间格式必须是 `HH:MM`（24小时制）
- 程序每分钟自动刷新一次
- 支持 Windows 10/11
